# StreamFi - Real-Time Crypto Payroll Platform

StreamFi is a decentralized blockchain application that automates salary streaming to employees in real-time using smart contracts.

## Features
- Real-time salary streaming per second
- Employee dashboard with balance tracking
- One-click token claiming
- Smart contract automation (Solidity)

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Ethers.js
- **Smart Contracts:** Solidity, Truffle, Ganache
- **Blockchain:** Ethereum-compatible networks
